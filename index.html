Get Weather Library

Library which returns the current weather in a requested city and country.

View project on GitHub
Watch our Project Gide on YouTube

What is the purpose of GET-WEATHER library ?

  • The library allows other developers to get and use weather data around the world.
  • You can easily get the current weather data of a required location by its city and country names.
  • This library refers to one RESTful web service, but as a developer you can add other libraries.

More about the GET-WEATHER library..

  • The library uses the RESTful web services from this website : [http://www.openweathermap.org/]
  • The library has been developed in the c# language.
  • The library has been developed in the Visual studio 2015 environment.
  • Unit testing was performed.
  • The library has been developed as part of the final project in the course 'Programming Languages'.

Design pattern implementation in the library :

  • Singleton
  • Factory

What is included in the GET-WEATHER library ?

  • A folder with all classes, interfaces and files that needed in order to use the library.
  • Library's Solution file to run on Visual Studio environment.
